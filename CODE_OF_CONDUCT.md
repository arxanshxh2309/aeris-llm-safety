# Code of Conduct

## Our pledge

We want this to be a welcoming, harassment-free project for everyone, regardless of background or experience level.

## Standards

Examples of behavior that contributes to a positive environment:
- Being respectful of differing viewpoints
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the project and the broader goal of safer AI systems

Examples of unacceptable behavior:
- Harassment of any kind
- Submitting contributions intended to weaponize the project (see `docs/safety_posture.md`)
- Using the project to attack systems you don't have permission to test

## Scope

This applies in all project spaces (issues, PRs, discussions) and when representing the project publicly.

## Enforcement

Report unacceptable behavior to the maintainer at conduct@example.com (replace with your real address). All complaints will be reviewed and investigated promptly and fairly.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
